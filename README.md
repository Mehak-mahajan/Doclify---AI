# Doclify AI 📚🤖

Doclify AI is an AI-powered study assistant that allows users to upload PDFs and interact with their study material using AI.

The application extracts text from uploaded documents, converts it into embeddings, stores it using vector search, and answers user queries based on the uploaded content.

---

# 🚀 Features

- Upload PDF documents
- Extract text from PDFs
- AI-powered chat with notes
- Semantic search using vector database
- Fast and accurate document retrieval
- Clean Streamlit interface
- Helps students revise efficiently
